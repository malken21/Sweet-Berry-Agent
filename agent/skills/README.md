# カスタムスキル作成ガイド

このディレクトリ（`agent/skills/`）は、AIエージェントの機能を拡張するためのカスタムスキル（Pythonスクリプト）を格納する場所です。

## スキルの仕組み
エージェント（OpenHands）は、起動時にこのディレクトリ内のPythonファイルをスキャンし、定義された関数をツールとして利用可能にします。

## スキルの作成方法
1. **Pythonファイルの作成**: `skills/` ディレクトリ内に `.py` ファイルを作成します。
2. **関数の定義**: エージェントに実行させたい処理を関数として定義し、適切なdocstringを記述してください。

### 実装例: `hello_skill.py`
```python
def say_hello(name: str) -> str:
    """
    指定された名前に挨拶を返します。
    
    Args:
        name: 挨拶する相手の名前
    """
    return f"こんにちは、{name}さん！Sweet Berry Agentへようこそ。"
```

## 注意事項
- スキルはエージェントのサンドボックス内で実行されます。
- 外部ライブラリが必要な場合は、あらかじめサンドボックス環境にインストールしておく必要があります。
- docstringの内容がエージェントによるツールの選択に影響するため、正確に記述してください。
